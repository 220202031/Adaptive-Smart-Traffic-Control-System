# Adaptive-Smart-Traffic-Control-System
Smart adaptive traffic management system with vehicle detection, pedestrian control, and dynamic signal timing using Arduino Uno.
## Project Objective
This project aims to develop a real-time adaptive traffic control system using vehicle density detection, pedestrian crossing management, and RTC-based night mode logic.

## Features
- Real-time vehicle density detection
- Adaptive traffic signal timing
- RTC-based night flashing mode
- Pedestrian crossing signal system
- Dynamic lane prioritization
- Fair traffic cycle scheduling

## Modules
1. Vehicle Detection Module
2. Adaptive Signal Timing Module
3. RTC Night Mode Module
4. Pedestrian Crossing Module
5. Traffic Light Control Module

## Hardware Components
- Arduino Uno / Mega
- HW201 IR Sensors
- DS3231 RTC Module
- LEDs
- Resistors
- Breadboard and jumper wires

## System Logic
The system continuously monitors vehicle density using IR sensors and dynamically adjusts green light durations according to traffic conditions. Each lane receives fair scheduling to prevent starvation while maintaining adaptive traffic optimization. During night hours, the system switches into flashing yellow mode using RTC time synchronization.
